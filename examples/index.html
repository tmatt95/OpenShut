Examples

Basic Usage

Options

Name Type Default Description
textHtml Boolean false Whether the output of the text contains html.
textOpen Text 'Open' Default text for when open
textClosed Text 'Closed' Default text for when closed
timeURL: Text '' Where to get the time from
Status Text 'open' Status, whether open or closed
loadingText: Text '' Enables you to set a message when the widget is loading
loadingHtml: Text 'false' Whether the loading text contains html.
timeDateGenerated: null null The time at which the display was generated

Shows the plugin running with only the required settings.
