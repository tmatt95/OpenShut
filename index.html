Basic Usage

Shows the plugin running with only the required settings.

<div id="basicos"></div>
<script type="text/javascript">
    $('#basicos').openShut(
	{
	    'days': {
		'monday': {'open': '00:00', 'closed': '00:00'},
		'tuesday': {'open': '00:00', 'closed': '00:00'},
		'wednesday': {'open': '00:00', 'closed': '00:00'},
		'thursday': {'open': '00:00', 'closed': '00:00'},
		'friday': {'open': '01:00', 'closed': '23:00'},
		'saturday': {'open': '07:00', 'closed': '16:00'},
		'sunday': {'open': '00:00', 'closed': '00:00'}
	    }
	}
    );
</script>

Excluded Dates

<div id="excludeddates"></div>
<script type="text/javascript">
    $('#excludeddates').openShut(
	{
	    'days': {
		'monday': {'open': '00:00', 'closed': '00:00'},
		'tuesday': {'open': '00:00', 'closed': '00:00'},
		'wednesday': {'open': '00:00', 'closed': '00:00'},
		'thursday': {'open': '00:00', 'closed': '00:00'},
		'friday': {'open': '01:00', 'closed': '23:00'},
		'saturday': {'open': '07:00', 'closed': '16:00'},
		'sunday': {'open': '00:00', 'closed': '00:00'}
	    },
	    datesClosed: [
		{
		    'start': '22/01/2014 12:00:00',
		    'end': '12/12/2014 23:59:59'
		}
	    ],
	}
    );
</script>

Options

Name Type Default Description
container Text 
'{text}'
 The container that contains the status
days Object 
{
    'monday': {'open': '00:00', 'closed': '00:00'},
    'tuesday': {'open': '00:00', 'closed': '00:00'},
    'wednesday': {'open': '00:00', 'closed': '00:00'},
    'thursday': {'open': '00:00', 'closed': '00:00'},
    'friday': {'open': '00:00', 'closed': '00:00'},
    'saturday': {'open': '07:00', 'closed': '16:00'},
    'sunday': {'open': '00:00', 'closed': '00:00'}
}
 The days of the week and when it is open / closed
datesClosed object 
[]
 The dates that the place is shut
status Text 
'open'
 Status, whether open or closed
textHtml Boolean 
false
 Whether the output of the text contains html
textOpen Text 
'Open'
 Default text for when open
textClosed Text 
'Closed'
 Default text for when closed
timeDateGenerated Date 
null
 The time at which the display was generated
